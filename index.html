The Pet Chooser Capstone Project

The Pet Chooser uses the Petfinder API and the Etsy API, each load a dog from their
respective sites and allow a user to choose a real dog to adopt from Petfinder
or allows the user to choose a stuffed animal dog from a pre selected Etsy store.
Selection happens based on breed and the user has a forward and backward button they can use to
refresh the selection until something they like appears.
The Pet Chooser was made with javascript, jquery and AJAX.

Order For Later

Order For Later is a web app that serves as a white board replacement for the compounding pharmacy
I work in. Each department requires different materials to be ordered throughout the week and
instead of having to write everything on a white board, when a department needs to order something
they can simply visits the web app address and add the requested material.
Order For Later then adds the material to a database that is then viewable once the page is refreshed.
A user has the ability to delete and item and highlight any material to indicate if an item is on back order
Order For Later uses, javascript, jquery, node.js and express and MongoDB for the Database structure.
